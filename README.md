### Criando arquivo de configuração
```
$ npm init -y
```
### Instalando biblioteca chalk que ajuda a localizar textos
```
$ npm install chalk
```
### Rodar o programa
```
$ node cli.js ./arquivos/texto1.md
```
ou
```
$ npm run cli validar
```
### Instalando framework de testes Jest
```
$ npm install --save-dev jest
```
